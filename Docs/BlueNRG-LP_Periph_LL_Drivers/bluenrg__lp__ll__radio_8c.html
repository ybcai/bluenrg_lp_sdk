void RADIO_SetChannelMap (uint8_t StateMachineNo, uint8_t *chan_remap)
 This routine sets the channel map. If channel map is not in use, do not use it. The chan_remap constant is a 37-bit vector, one bit per channel from 0 to 36, with the LSB corresponding to the channel 0. If the corresponding bit is set, the channel is in use; if it is cleared, there is an automatic remap to another channel, as defined in the Bluetooth Low Energy specification. Setting all bits of the array to one, disables the channel remapping, and this is the mode that should be used if the Bluetooth channel remap is not in use. More...
