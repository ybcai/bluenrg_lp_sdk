142  /*---------------------------- I2Cx CR1 Configuration ------------------------
143  * Configure the analog and digital noise filters with parameters :
144  * - AnalogFilter: I2C_CR1_ANFOFF bit

155 
156  /* Enable the selected I2Cx Peripheral */
