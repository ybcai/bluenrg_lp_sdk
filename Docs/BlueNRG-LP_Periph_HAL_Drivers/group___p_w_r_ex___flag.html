Detailed Description

Elements values convention: 0000 0000 00XX YYYYb

  • YYYY: Flag position in the XX register (4 bits)
  • XX : Status register (2 bits)
    • 01: SR1 register
    • 10: SR2 register
    • 11: C2_SCR register The only exception is PWR_FLAG_WUF, encompassing all wake-up flags and set to PWR_SR1_WUF.
