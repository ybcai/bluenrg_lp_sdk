Definition: bluenrg_lp_hal_rcc.h:176

Macro to configure the External High Speed oscillator (HSE).

Note
Transition HSE Bypass to HSE On and HSE On to HSE Bypass are not supported by this macro. User should request a transition to HSE Off first and then HSE On or HSE Bypass.
After enabling the HSE (RCC_HSE_ON), the application software should wait on HSERDY flag to be set indicating that HSE clock is stable and can be used to clock the RC64MPLL.
HSE state can not be changed if it is used directly or through the RC64MPLL as system clock. In this case, you have to select another source of the system clock then change the HSE state (ex. disable it).
The HSE is stopped by hardware when entering DEEPSTOP and SHUTDOWN modes.
Parameters
<strong>STATE</strong>specifies the new state of the HSE. This parameter can be one of the following values:
  • RCC_HSE_OFF Turn OFF the HSE oscillator, HSERDY flag goes low after 6 HSE oscillator clock cycles.
  • RCC_HSE_ON Turn ON the HSE oscillator.
Return values
None

#define __HAL_RCC_LSI_ENABLE ( )    LL_RCC_LSI_Enable()

Macros to enable or disable the Internal Low Speed oscillator (LSI).

Note
After enabling the LSI, the application software should wait on LSIRDY flag to be set indicating that LSI clock is stable and can be used to clock the IWDG and/or the RTC.
LSI can not be disabled if the IWDG is running.
When the LSI is stopped, LSIRDY flag goes low after 6 LSI1 oscillator clock cycles.
Return values
None
