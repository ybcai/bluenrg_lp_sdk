void NVIC_SetVectorTableOffset ( uint32_t  offset)

Set the vector table offset address. On reset, the Cortex-M0+ vector table is fixed at address 0x00000000. The software may relocate the vector table address to a different memory location, in a range 0x00000000 to 0xFFFFFF80 in multiples of 256 bytes through the Vector Table Offset Register (VTOR) located in the Cortex-M0+ registers area.

Parameters
offsetTable offset address Remember to modify the linker file according the new offset. i.e. if the offset is 0x600, place the vector table in the linker file at address 0x10040600
